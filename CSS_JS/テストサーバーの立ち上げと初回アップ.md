# テストサーバーの立ち上げと初回アップ

タイプ: フロー

## 利用シーン（あれば画像）

- タイトルのとおり

## やること

- レンタルサーバー

  1. サブドメインを追加
     1. ConoHa の場合：サーバ管理 → ドメイン → サブドメイン
  2. FTP 情報を取得。
     1. ConoHa の場合：サイト管理 →FTP
     2. ユーザー名は、`dev~`
     3. パスワードは Bitwarden で生成。
     4. Bitwarden の名前は`FTP_{site name}_{test or production}`
  3. 必要であれば WordPress をインストール
     1. ConoHa の場合：サイト設定 →WordPress
     2. パスワードは、ローカルのワードプレスと同じにする。
  4. 情報はこれくらい揃えてメモすればよし。Bitwarden にも FTP 情報と WP ログイン情報は保存忘れず。

     ```markdown
     ### テストサーバー情報

     - サイト：https://test-shimazugumi-corporate.saki-llc.com/
     - FTP 情報
       - ID：`dev@test-shimazugumi-corporate.saki-llc.com`
       - PASS：`Be3zz9r3EmAEUo!`
     - テストサーバーワードプレスログイン情報
       - http://www.test-shimazugumi-corporate.saki-llc.com/wp-admin/
       - ID：`shimazugumi_corporate_dev`
       - PASS：`shimazugumi_corporate_dev0!`
     ```

- Local
  1. WordPress ファイルをエクスポート
- テストサーバーのワードプレス
  1. ログイン
  2. All-in-One WP Migration  をインストール
     1. https://drive.google.com/file/d/1LJ4ZMhtOtKC37JpBZLEgcfNOI4DzOwDK/view?usp=sharing
  3. エクスポートしたファイルをインポート
