### CSS Updates for Accordion Component

To enhance animation performance, the following `will-change` properties have been added:

```css
.accordion__content {
    will-change: grid-template-rows;
}

.accordion__content-inner {
    will-change: padding, opacity, transform;
}

.accordion__icon {
    will-change: transform, color;
}
```

These changes aim to optimize the rendering performance of the accordion component's animations.
