# HTML・SEO・コンテンツ チェック用プロンプト

タイプ: プロンプト

## テキスト・コンテンツの基本チェック

```
サイト全体のテキストコンテンツを確認し、以下の点について問題がないかチェックしてください。

1. 誤字脱字、スペルミスがないか（漢字、ひらがな、半角全角の統一など）
2. 意図しない日付の表記ゆれがないか（2023年1月1日、2023.01.01 などが混在していないか）
3. 「Lorem ipsum」や「ダミー」「テスト」などの制作過程のダミーテキストが残っていないか
4. 開発環境のローカルパス（localhost, 192.168... や file:// など）がコード内に残っていないか
```

## メタデータとSEO設定のチェック

```
headタグ内およびサイト全体のSEO設定について、以下を確認してください。

1. titleタグ、meta descriptionが各ページに適切に設定されているか（ページごとにユニークか）
2. OGPタグ（og:title, og:image, og:description, og:url, twitter:card 等）が適切に設定されているか
3. faviconが設定されているか
4. canonical（正規化URL）が指定されているか（推奨）
5. 文字コード（UTF-8）が指定されているか
6. 構造化データ（JSON-LD）が記述されている場合は、構文エラーがないか
```

## HTMLマークアップと構造のチェック

```
HTMLの構造とタグの使用について、コードベース全体をチェックしてください。

1. 見出しタグ（h1〜h6）の階層構造は正しいか（h1は原則1ページに1つ、レベルが飛んでいないか）
2. コンテンツの内容に合わせて適切なセマンティックHTMLタグ（main, section, article, nav, header, footer, aside 等）が使用されているか
3. timeタグまたはdate-time属性が、日付や時刻を表す箇所で適切に使用されているか
4. 住所情報が表示されている場合、addressタグが使用されているか
5. HTMLの予約文字（<, >, &, ", '）が適切にエスケープ（文字参照）されているか
6. 不要なコメントアウト（TODO: や作業メモなど）が残っていないか
```

## 画像とリンクの基本チェック

```
画像とリンクの実装について確認してください。

1. すべてのimgタグにalt属性が設定されているか（装飾用画像は alt="" となっているか）
2. 内部リンク、外部リンクが正しく設定されており、リンク切れの可能性がないか（href="#" や href="" のまま放置されていないか）
3. 引用画像がある場合、引用元の表記やblockquote/citeタグが使用されているか
4. 外部サイトへのリンクで target="_blank" を使用する場合、セキュリティ対策（rel="noopener" ※最近のブラウザでは暗黙的に適用されるが、要件により明示が必要な場合を確認）が考慮されているか
```
