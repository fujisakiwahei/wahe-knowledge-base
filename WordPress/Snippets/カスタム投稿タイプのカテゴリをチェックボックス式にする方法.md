## やること

`function.php`にコードを追加

```php
/* ===============================================
//カスタムタクソノミーをチェックボックスで選択できるようにする
 =============================================== */

//  ブログ
function change_term_to_checkbox_blogs() {
  $args = get_taxonomy('blogs_category'); //★カスタムタクソノミー名
  $args->hierarchical = true; //Gutenberg用
  $args->meta_box_cb = 'post_categories_meta_box'; //Classicエディタ用
  register_taxonomy('blogs_category', 'blogs', $args); //★カスタムタクソノミー名、カスタム投稿タイプ名
}
add_action('init', 'change_term_to_checkbox_blogs', 999);
```
