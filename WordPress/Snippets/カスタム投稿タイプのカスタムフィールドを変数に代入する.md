# カスタム投稿タイプのカスタムフィールドを変数に代入する

タイプ: スニペット

## 利用シーン（あれば画像）

- タイトルの通り

## コード

- 変数にカスタムフィールドの値を代入（下記は一例。実際に項目によって変動）

```php
$cover_image_id = post_custom('cover_image');
$cover_image_url = wp_get_attachment_url($cover_image_id);
$name = post_custom('name');
$link = get_the_permalink();

$post_id = get_the_ID();
$terms = get_the_terms($post_id, '{タクソノミー名}');
```
