# カスタム投稿タイプのタグを取得する

タイプ: スニペット

## 利用シーン（あれば画像）

- 配列で取得して、タグを表示させたい時など

![image.png](%E3%82%AB%E3%82%B9%E3%82%BF%E3%83%A0%E6%8A%95%E7%A8%BF%E3%82%BF%E3%82%A4%E3%83%97%E3%81%AE%E3%82%BF%E3%82%B0%E3%82%92%E5%8F%96%E5%BE%97%E3%81%99%E3%82%8B/image.png)

## コード

```php
<?php
	$terms = get_the_terms(get_the_ID(), '{取得したいタグのスラッグ}');
?>

<ul class="workDataBottom">
  <?php if ($terms) : ?>
      <?php foreach ($terms as $term) : ?>
          <li class="workDataTag">
              <?php echo $term->name; ?>
          </li>
      <?php endforeach; ?>
  <?php endif; ?>
</ul>
```
