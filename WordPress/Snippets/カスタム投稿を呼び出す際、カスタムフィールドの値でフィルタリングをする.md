# カスタム投稿を呼び出す際、カスタムフィールドの値でフィルタリングをする

タイプ: スニペット, フロー

## 利用シーン（あれば画像）

- タイトルの通り
- `WP_Query`の`meta_query`引数を使用することで、カスタムフィールドの値に基づいて投稿を絞り込むことができる

## コード

- `meta_query` を追加する
    - **`key`**: フィルタリングしたいカスタムフィールドの名前を指定します。例のコードでは 'genre' というフィールドが指定されています。
    - **`value`**: フィルタリングする値を指定します。例のコードでは 'newly-built' という値で絞り込んでいます。
    - **`compare`**: 比較演算子を指定します。例のコードでは '=' (等しい)を使用していますが、他にも '!='（等しくない）、'>'（より大きい）、'<'（より小さい）などが使用できます。
    
    これらのパラメータを使用することで、カスタムフィールドの値に基づいて投稿を絞り込むことができます。
    
    ```php
    'meta_query' => array(
        array(
            'key' => 'genre',
            'value' => 'newly-built',
            'compare' => '='
        )
    )
    
    ```
    

## 注意点
