# 表示しているページに応じたロゴの切り替え

タイプ: スニペット

## 利用シーン（あれば画像）

- トップページならh1、下層ページならpで囲む

## コード

```php
<div class="headerLogo">
    <?php
    $logo_link = '<a href="' . home_url() . '"><img src="' . get_template_directory_uri() . '/assets/images/common/logo.svg" alt="mind palaceロゴ"></a>';
    if (is_front_page()) : ?>
        <h1><?php echo $logo_link; ?></h1>
    <?php else : ?>
        <p><?php echo $logo_link; ?></p>
    <?php endif; ?>
</div>
```

## 注意点

- 画像パスとaltは必ず書き換える
