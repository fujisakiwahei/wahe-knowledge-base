# タクソノミーのリストを表示する

タイプ: スニペット

## 利用シーン（あれば画像）

- カテゴリやタグを表示させる場合

## コード

### 前提

タームを、変数に入れる必要あり

```php
$post_id = get_the_ID();
$terms = get_the_terms($post_id, '{タクソノミー名}');
```

### カテゴリ・タグが1つの場合

```php
<p class="">
<?php foreach ($terms as $term) : ?>
    <?php echo $term->name; ?>
<?php endforeach; ?>
</p>
```

### カテゴリ・タグが複数の場合

```php
<ul class="">
<?php foreach ($terms as $term) : ?>
    <li class=""><?php echo $term->name; ?></li>
<?php endforeach; ?>
</ul>
```
