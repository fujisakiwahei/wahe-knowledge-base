# ギャラリーの画像を取得する

タイプ: スニペット

## 利用シーン（あれば画像）

- カスタム投稿タイプで、ギャラリーの画像を管理するとき

![image.png](%E3%82%AE%E3%83%A3%E3%83%A9%E3%83%AA%E3%83%BC%E3%81%AE%E7%94%BB%E5%83%8F%E3%82%92%E5%8F%96%E5%BE%97%E3%81%99%E3%82%8B/image.png)

## コード

1. ギャラリーの配列を定義
2. 画像の URL を、for でそれぞれ格納

   ```php
   $gallery_image_urls = [];
   for ($i = 1; $i <= 20; $i++) {
       $image_field = 'gallery_image' . str_pad($i, 2, '0', STR_PAD_LEFT);
       $image_id = get_field($image_field);
       $gallery_image_urls[$i] = $image_id;
   }
   ```

3. 画像をリストで表示。

   ```php
   <ul class="workGalleryList">
       <?php for ($i = 1; $i <= 20; $i++) : ?>
           <?php if ($gallery_image_urls[$i]) : ?>
               <li class="workGalleryItem">
                   <img src="<?php echo $gallery_image_urls[$i]; ?>" alt="Gallery Image" />
                   <p class="workGalleryDescription"><?php echo $gallery_descriptions[$i]; ?></p>
               </li>
           <?php endif; ?>
       <?php endfor; ?>
   </ul>
   ```

## 注意点

- カスタムフィールド名は、配列の添字と一致させるために`gallery_image1`のように数字のみにした方が、コードがシンプルになります。（`gallery_image01`じゃない）
