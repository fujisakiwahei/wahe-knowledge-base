# タクソノミーのアーカイブページ作成

タイプ: スニペット, フロー

## 利用シーン（あれば画像）

- タイトルの通り

## コード

### 手順

1. `taxonomy-{タクソノミー名}.php`を作成
2. `WP_Query` に、`tax_query`を追加する。そこで表示する投稿を限定

   ```php
   <?php
   $term = get_queried_object();
   $paged = (get_query_var('paged')) ? get_query_var('paged') : 1;
   $args = array(
       'post_type' => 'blogs',
       'posts_per_page' => 8,
       'orderby' => 'date',
       'order' => 'DESC',
       'paged' => $paged,
       'tax_query' => array(
           array(
               'taxonomy' => 'blogs_category',
               'field' => 'slug',
               'terms' => $term->slug,
           ),
       ),
   );

   $query = new WP_Query($args);
   ?>
   ```
