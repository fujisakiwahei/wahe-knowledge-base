# All-in-One SEO 基本設定

タイプ: フロー
技術・ツール: WordPress

WordPressプラグイン「All-in-One SEO (AIOSEO)」を使用し、サイトのSEO基礎設定およびSNSシェア設定（OGP）を管理するための手順書です。

---

## 1. 全体設定フロー（初回・メンテナンス時）

サイト全体に関わる共通設定の確認手順です。

- [ ]  **① メタ情報（Title, Description）の基本設定**
    - **確認場所:** [All in One SEO] > [検索外観]
    - **全体設定:** トップページのタイトル・説明文が入力されているか。
    - **コンテンツタイプ:** 「投稿」「固定ページ」の設定が適切か（タイトルタグ、抜粋設定など）。
- [ ]  **② Favicon（ファビコン）の設定**
    - **確認場所:** [設定] > [一般]
    - **設定:** 「サイトアイコン」に512x512px以上の画像が設定されているか。
- [ ]  **③ OGP（SNS設定）の全体設定**
    - **確認場所:** [All in One SEO] > [ソーシャルネットワーク]
    - **Facebook:** 「OGマークアップ」が有効か。「デフォルトの投稿画像（代替画像）」が設定されているか。
    - **Twitter:** 「Twitterカード」が有効か。「カードタイプ」が「大きな画像付きのまとめ」になっているか。
        
        <aside>
        
        **Xカード設定（上側のブロック）**
        └ サイト全体のデフォルトルール
        - 各投稿・固定ページの基本挙動
        - アイキャッチがないときの代替画像
        
        **ホームページ設定（下側のブロック）**
        └ トップページURL専用の上書き設定
        - トップだけ特別なタイトル・説明・画像にできる
        
        </aside>
        
- [ ]  **④ SNS URLの設定**
    - **確認場所:** [All in One SEO] > [ソーシャルネットワーク]
- [ ]  **⑤ 重複・エラーチェック（localでは不可）**
    - **確認場所:** [All in One SEO] > [SEO分析] > [チェックリスト]
    - **確認:** 重大な問題として「重複したタイトル」「重複したメタディスクリプション」が警告されていないか。

---

## 2. ページ単位の対応フロー（記事公開・更新時）

記事や固定ページを作成・更新する際のルーチンワークです。

- [ ]  **① 管理画面での編集・確認（AIOSEO設定）**
    - **確認場所:** 記事編集画面下部 or サイドバーの [AIOSEO設定]
    - **一般タブ:**
        - タイトルは適切か（自動 or 手動）。
        - メタ説明文に記事の要約（120文字前後）が入っているか。
    - **ソーシャルタブ:**
        - Facebook / Twitter のプレビュー画像が意図したものになっているか。
        - 必要に応じて「画像ソース」でアイキャッチ以外の画像を指定する。
- [ ]  **② OGPの表示確認（外部ツール）**
    - **確認場所:** [Facebook シェアデバッガー](https://developers.facebook.com/tools/debug/) 等
    - **確認:** 記事URLを入力し、画像・タイトル・説明文が正しく表示されるか確認する。エラーが出る場合は「もう一度スクレイピング」を実行する。
