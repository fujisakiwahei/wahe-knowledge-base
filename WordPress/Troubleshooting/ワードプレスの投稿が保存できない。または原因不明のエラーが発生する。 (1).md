# ワードプレスの投稿が保存できない。または原因不明のエラーが発生する。 (1)

タイプ: トラブルシューティング

## 現象

`{code: 'invalid_json', message: 'The response is not a valid JSON response.'}` というエラーが出て、ワードプレスの投稿の保存に支障が出る。

## 原因

`function.php` の冒頭に、明らかに不要なコードが紛れ込んでいた。

## 確認手順

1. `function.php`に不要なコードが入っていないか確認する
2. その他のphpファイルも同様に確認

## 解決方法

1. 不要なコードを削除
