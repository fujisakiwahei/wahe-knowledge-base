# 配列の値を〇〇件だけ表示したい

タイプ: スニペット

## 利用シーン（あれば画像）

- 配列から最初の指定件数（この場合は8件）のみを取得したい場合に使用
- APIレスポンスや大きな配列から、表示する件数を制限する際に便利

```tsx
// APIからデータを取得
const { data } = await useFetch('/api/works')

// computed propertyを使用して最初の8件のみを取得
const limitedWorks = computed(() => data.value?.contents.slice(0, 8))

```

## コードの説明

- **Array.slice()**メソッドを使用して配列の一部を切り出し
- **computed()**を使用して、元データが変更された場合に自動的に再計算
- slice(0, 8)は、インデックス0から7までの要素（計8件）を取得

## 注意点

- slice()は元の配列を変更せず、新しい配列を返す
- データが8件未満の場合は、全件取得される
- TypeScriptを使用して型安全性を確保している
